
BACKEND----

DB - Contiene la conexión central a MySQL.
Helper - Funciones para sanitizar inputs y devolver JSON.
Productos - Endpoint para listar productos, buscar y filtrar.
Carrito - Endpoint para agregar/quitar productos y ver carrito.
Pedido - Endpoint para marcar compra como realizada.
Login - Endpoint para autenticación básica de clientes.


FRONTEND----

JS/app - Toda la logica para agilizar.

ADMIN/login.html - formulario de acceso administrador.
ADMIN/login.php - valida credenciales contra tabla TUsuarioAdmin.
ADMIN/index.html - dashboard con enlaces a productos y compras.
ADMIN/productos.html - listado y formulario CRUD.
ADMIN/productos.js - llamadas AJAX a admin_productos.php.
ADMIN/compras.html - muestra pedidos registrados.
